# Build a Basic Website

If you want to **make a website**, start with one HTML file, one CSS file, and one JavaScript file. This is the best beginner setup for a **basic website**, **starter site**, or **starter HTML CSS JS project** because you can see exactly what each file does.

## Smallest beginner website setup

```text
site/
  index.html
  styles.css
  app.js
```

Use `index.html` for the page structure, `styles.css` for layout and colors, and `app.js` for any interactive behavior. Do not start with a framework if your goal is just to build a simple website.

## Minimal beginner website flow

1. Make the page structure in HTML.
2. Add spacing, colors, and layout in CSS.
3. Add small interactions in JavaScript only where needed.

## Beginner website page structure

A good first website page usually has:
- header
- navigation
- hero section
- main content section
- contact or call-to-action section
- footer

## Good rule

Start plain first. A plain website with clean HTML, CSS, and JS is better than jumping straight into a complex stack before you understand page structure.
