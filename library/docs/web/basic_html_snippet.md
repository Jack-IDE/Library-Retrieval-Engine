# Basic HTML Snippet

If someone asks for a **basic HTML snippet**, give them a complete valid page, not a fragment with missing `<head>` or `<body>` tags.

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Website</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <header>
    <h1>Hello world</h1>
    <p>This is my basic website starter.</p>
  </header>
  <script src="app.js" defer></script>
</body>
</html>
```

This is the right answer for queries like:
- basic html snippet
- simple html page
- starter html page
- basic website html
