# Multi-Page Site Starter

A **multi-page site** is a good fit when the content is mostly static and each page has a clear purpose.

Use a simple structure like this:

```text
site/
  index.html
  about.html
  contact.html
  styles.css
  app.js
```

This is better than a client-side router when you just need a small business site, portfolio, brochure site, or documentation-style website.

Common beginner queries this helps with:
- multi-page site
- basic website with about and contact pages
- small static website
