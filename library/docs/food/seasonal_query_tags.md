# Seasonal and Vibe Food Tags

Use explicit food tags when the query is broad, seasonal, mood-based, or beginner-oriented.

Core tags:
- season: autumn, winter, spring, summer, all-season
- vibe: cozy, comforting, simple, plain, quick, fun, sweet, savory, classic
- meal type: breakfast, lunch, dinner, snack, dessert
- difficulty: easy, medium, advanced

Good query-to-tag mappings:
- autumn soup -> season: autumn + vibe: cozy + meal type: lunch/dinner
- cozy soup -> vibe: cozy, comforting
- eggs and toast -> vibe: plain, simple + meal type: breakfast
- basic breakfast -> vibe: simple, quick + meal type: breakfast
- fun bread for autumn -> season: autumn + vibe: fun, cozy, sweet
- simple bread -> vibe: simple + meal type: breakfast or snack
- comfort food -> vibe: comforting, classic, cozy
